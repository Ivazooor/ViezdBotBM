# Бот выездных фотоотчётов (Бизнесмакет)

Telegram-бот для отдела выездов: сотрудник создаёт фотоотчёт прямо в переписке с ботом
(тип → чек-лист → проект/дата/ответственный/комментарий → фото и видео), а бот пересылает
всё в рабочий чат.

Файлы **до 2 ГБ** — потому что их присылает приложение Telegram сотрудника, а бот лишь
пересылает их по `file_id` (`copyMessage`). Свой Bot API сервер и `api_id` не нужны.

## Настройка и запуск

См. подробную инструкцию: [BOT_SETUP.md](BOT_SETUP.md).

Кратко — три переменные окружения:
- `TELEGRAM_BOT_TOKEN` — токен бота из @BotFather;
- `TELEGRAM_CHAT_ID` — id рабочего чата/группы;
- `ALLOWED_USER_IDS` — Telegram ID сотрудников через запятую.

```bash
npm install
cp .env.example .env   # вписать значения
npm start
```
