# ViezdBot БМ — Telegram-бот выездных фотоотчётов

Бот для отдела выездов «Бизнесмакет». Сотрудник создаёт фотоотчёт прямо в переписке с ботом
(тип → чек-лист → проект/дата/ответственный/комментарий → фото и видео → сводка → отправка),
а бот пересылает всё в рабочий чат.

- **Файлы до 2 ГБ** — их присылает приложение Telegram сотрудника, бот лишь пересылает по `file_id`.
- **Работает на обычном PHP-хостинге** (RU-CENTER) через webhook — отдельный сервер и `api_id` не нужны.
- Доступ — только у сотрудников из списка `allowed_ids`.

## Состав

- `bot.php` — весь бот (webhook-обработчик, FSM, отправка).
- `config.sample.php` — шаблон настроек (скопировать в `config.php`).
- `BOT_SETUP.md` — пошаговая установка на RU-CENTER.

## Установка

См. [BOT_SETUP.md](BOT_SETUP.md). Кратко: создать `config.php`, залить `bot.php` + `config.php`
на хостинг, привязать webhook через `setWebhook`.
